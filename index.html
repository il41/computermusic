HOW IT WORKS

Computer Music is a music player and album that creates its tracks with user input and machine learning interaction. Streamlined and simplified machine learning software takes initial user input to begin a track. User interaction will build and develop aspects of the song with every Replay. Designed only with fundamental learning procedures and some simple timing frameworks for songwriting (that could be learned differently later on with use), the music is created with user guidance only, keeping outside influences and biases minimal.

The intention of this is to unhinge western musical conventions and artificial intelligence conventions from music making and AI interaction. New types of objective music can be created in a class of their own. Harmonic, rhythmic, structural, and stylistic musical possibilities can be explored with the use of this player. The possibilities allowed by this range from creating and inspiring new music to emulating existing musical conventions. Relying only on user input as content, an interesting musical reflection of the user can be generated.
